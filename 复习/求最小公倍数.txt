题目描述
正整数A和正整数B 的最小公倍数是指 能被A和B整除的最小的正整数值，设计一个算法，求输入A和B的最小公倍数。

输入描述:
输入两个正整数A和B。

输出描述:
输出A和B的最小公倍数。

示例1
输入
复制
5 7
输出
复制
35

import java.util.*;
public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();
        if(a > b){
            int tmp = a;
            a = b;
            b = tmp;
        }
        for(int i = b;i <= a*b;i++){
            if((i%a==0) && (i%b==0)){
                System.out.println(i);
                break;
            }
        }
    }
}
